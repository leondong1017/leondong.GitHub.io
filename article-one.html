如何搭建你的个人设计网站 | How to setup your own portfolio website

步骤 | Step

  1. 第一步先去购买域名，可选项GoDaddy / 阿里云 / 腾讯yun（后面二者缺点，后面弄DNS的时候需要ICP备案，需要时间）。
  2. 第二步找DNS解析商，可以阿里云/腾讯云，但还是上述问题。如果用GitHub Page来搭就不需要。
  3. 第三步GitHub页面实现，先下载GitHub Desktop端，这样可以本地进行版本管理及同步，很方便。使用Espresso（Mac端）进行html和css的编写。网页模版可以在html5up直接选一个。当然也可以不用espresso，其他css工具推荐
  4. 第四步页面设计。这个就是设计师的主场了。html&css语言手册

GitHub Page实现路径

我这边是用GoDaddy + GitHub Page实现的～

  • 注册GitHub账号
  • 到个人页面，Create a new repository
  • 在选项中间注意repository name最好和你username一致，否则会出现username.github.io/username2.github.io这种情形，在设置DNS时候比较烦
  • create后，进入Setting页面，在左侧面板选择Pages，在右侧可以看到你的网页链接，复制下来，去DNS Checker，然后搜索你复制的链接，看看它的ip是啥（xxx.xxx.xxx.xxx），把四个选项全粘下来（但是用ip选项也会有弊端，还是直接在上一步保持一致可以减少麻烦）
  • 去GoDaddy的DNS管理页面，在DNS记录卡片中，新增，类型选择A，名称填@，后面填上一步粘下来的ip*4。如果GitHub Page设置没取不一样的Username，你也可以直接新增，类型CNAME，名称填blog，后一步把你github page的地址粘过来
  • 在Github Page的Setting页面的Pages中，右下有个Custom Domain，在这里把你在购买的域名复制过来，确认，等待解析，该框转完圈圈以后+上方显示绿勾即成功
  • 设计你的个人页面呗～
